# myportfolio
A data analyst is responsible for collecting, analysing and interpreting data in order to produce a set of insights from cleaned dataset that are useful to organisations and businesses so as to help them make important decisions through establishment of important KPI metrics to measure effectiveness of business decisions.
I help businesses Identify areas within the business that can be made more efficient or improved, monitor data quality, structure large data sets to find patterns and actionable insights that are relevant to business objectives. Use graphs / charts and other methods to visualise the collected data, create reports for internal teams and external clients or stakeholders.

# Tools
- Ms Excel - Data Cleaning, Pivot tables, Data Visualisation
- PowerBI - Data Visualisation
- SQL - Using advanced queries to extract useful insights
